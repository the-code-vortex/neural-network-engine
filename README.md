# neural-network-engine

# Neural Network Engine from Scratch

A feedforward neural network built using only NumPy — 
zero ML frameworks.

## Tech Stack
Python, NumPy, Matplotlib, Flask, Render

## Features
- Forward and backpropagation implemented manually
- Mini-batch gradient descent from scratch
- Dropout regularization and learning rate scheduling
- Trained and evaluated on MNIST dataset
- Live Flask web app for handwritten digit prediction
